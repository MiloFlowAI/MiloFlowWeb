# MiloFlow Website

Static landing page for MiloFlow, ready for GitHub Pages. The page positions
MiloFlow as an iPhone-first personal AI memory workspace: a private agent that
can remember over time, use a user-owned knowledge vault, and turn conversations
into reminders, notes, and action items. It mirrors the current app model:
free Milo chat, Local Gemma, Gemini Cloud with a bring-your-own API key, Pro
Gemini Live/workspaces at $6.99 per month or $59.99 per year, Max hosted
Gemini at $16.99 per month or $149.99 per year, local vault memory, optional
smarter recall, and on-device-first storage.

## Files

- `index.html` - landing page
- `privacy.html` - privacy notes for App Store/support links
- `support.html` - support and troubleshooting contact page
- `styles.css` - responsive site styling
- `404.html` - GitHub Pages fallback page
- `assets/brand/` - copied MiloFlow logo and app icon assets

## Publish With GitHub Pages

1. Create a GitHub repository for the site.
2. Push this folder to the repository.
3. In GitHub, open **Settings** -> **Pages**.
4. Choose **Deploy from a branch** and select `main` / root.
5. After buying a custom domain, add a `CNAME` file containing the domain, for example:

```text
miloflow.ai
```

Then configure DNS records at the domain registrar for GitHub Pages.
