Support

How can we help?

Send bug reports, TestFlight requests, privacy questions, and account questions to MiloFlow support.

Email support Read privacy notes

Useful details

For bugs, include your device model, iOS version, selected plan, selected provider, whether you were using voice or text, and what happened right before the issue.

Cloud access

Pro cloud features use your own Google API key. Max adds hosted Gemini so you do not need to set up an API key for included usage. MiloFlow support cannot view or recover a key you provide.

Subscriptions

Pro is $6.99 per month or $59.99 per year. Max is $16.99 per month or $149.99 per year and includes 150 hosted Live minutes, 1,000 hosted text and tool actions, and 100 hosted web-search answers per month.

Vault memory

You can refresh changed vault files, delete the local memory index, or forget the selected vault folder from MiloFlow settings.

Apple Watch

The watch companion is a Gemini Live remote. Talk from your watch while iPhone handles tools, memory, history, transcript, and audio output.
