Privacy

Privacy Policy

MiloFlow is designed around on-device notes, explicit cloud use, and visible memory controls.

Last updated: May 23, 2026

Local storage

MiloFlow stores app records on your device, including chat history, workspace records, reminders, settings, and local memory index data. If you connect a Markdown vault, MiloFlow can write readable Markdown files to that selected folder.

Cloud model use

Gemini Cloud, Gemini Live, and optional smarter recall can use either a Google API key that you provide or, on Max, MiloFlow hosted Gemini. Requests may include the text, audio, images, notes, and tool context needed for the feature you use.

AI Memory

Vault search can stay on this device. Optional smarter recall may send relevant note text and search queries to Google through your key or MiloFlow hosted Gemini. You can delete the memory index or forget the selected vault folder from settings.

Subscriptions and hosted usage

Subscriptions are handled through Apple's App Store. MiloFlow may verify StoreKit entitlements with a backend and keep hosted-usage counters for Max limits. Usage counters are kept separately from the raw content of your conversations.

Incognito

Incognito Milo is for conversations that should not carry forward. It skips saved history, vault writes, memory recall, archives, and carry-over context.

Apple services

MiloFlow can use Apple Speech, local audio playback, Apple Reminders, WatchConnectivity, and system diagnostics when you enable related features or grant permissions.

Contact

Questions or deletion requests can be sent to admin@miloflow.ai.
